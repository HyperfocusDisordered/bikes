## Описание модулей

### Core
- **core.cljd** - точка входа, запускает приложение
- **app.cljd** - главное приложение с навигацией и темой

### Screens
- **home.cljd** - главный экран с быстрыми действиями
- **qr_scanner.cljd** - сканирование QR кода на байке
- **bike_rental.cljd** - управление арендой (старт/стоп)

### Components
- **pwa_install.cljd** - промпт для установки PWA

### Services
- **api.cljd** - HTTP клиент для взаимодействия с backend API
- **bluetooth.cljd** - сервис для работы с Bluetooth блокировщиками

### State
- **app_state.cljd** - глобальное состояние (текущая аренда, байк, пользователь)

### Utils
- **helpers.cljd** - вспомогательные функции (форматирование времени, валидация)
