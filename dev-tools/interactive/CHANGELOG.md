## Как проверить

1. Запустите: `cd dev-tools/interactive && ./start.sh`
2. Откройте: http://localhost:8000/dev-tools/interactive/index.html
3. Откройте консоль браузера (F12)
4. Кликайте на элементы дерева
5. Проверьте что нет ошибок в консоли

## Если все еще есть ошибки

1. Откройте консоль браузера (F12)
2. Скопируйте текст ошибки
3. Проверьте что сервер запущен из корня проекта
4. Убедитесь что URL правильный: `/dev-tools/interactive/index.html`
